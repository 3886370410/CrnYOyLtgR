# Python大数据景点人流分析系统

## 核心功能
- 登录注册
- 旅游景点详情查看
- 人流量预测
- 景区数据可视化
- 景点人流量分析大屏
- 用户管理
- 旅游景点管理

## 适用场景
- 旅游景区管理
- 旅游数据分析
- 人流预测与调控
- 旅游市场研究

## 技术特点
- 基于Python开发，使用Django框架
- 数据存储采用MySQL
- 前端界面使用HTML、CSS、JavaScript
- 提供详细文档和PPT，方便用户快速上手
